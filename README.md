# Dall-E Supercharged

A helper script leveraging ChatGPT-4 to generate powerful prompts for Dall-E 3.

Once generated, it will automatically open a browser tab with your image.

```bash
OPENAI_TOKEN=abcd1234 deno run index.ts A gigantic shiba dog in the city
```

Get a token [here](https://platform.openai.com/api-keys).
